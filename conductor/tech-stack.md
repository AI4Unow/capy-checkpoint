# Technology Stack: Mathie

## Core Technologies
*   **Language:** [TypeScript](https://www.typescriptlang.org/) (Primary for all development)
*   **Frontend Framework:** [React 19](https://react.dev/) (UI components and game integration)
*   **Platform Framework:** [Next.js 16](https://nextjs.org/) (Web platform, routing, and server-side logic)
*   **Game Engine:** [Phaser 3.90](https://phaser.io/) (Core game mechanics and rendering)

## UI & Styling
*   **CSS Framework:** [Tailwind CSS 4](https://tailwindcss.com/)
*   **Post-processing:** [PostCSS](https://postcss.org/)

## State Management & Data
*   **State Management:** [Zustand](https://zustand-demo.pmnd.rs/) (Lightweight, reactive state)
*   **Backend/Database:** [Firebase](https://firebase.google.com/) (Authentication, Cloud Firestore, Firebase Admin SDK)

## Infrastructure & Tooling
*   **Build Tool:** [Vite](https://vitejs.dev/) (For the standalone game client)
*   **Testing:** [Vitest](https://vitest.dev/) (Unit and integration testing)
*   **Scripting/Execution:** [tsx](https://tsx.is/) (Running TypeScript scripts directly)
