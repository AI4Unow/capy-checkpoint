# Phase 01: Design Guidelines

## Context
Establish the visual language for Capy-Checkpoint.

## Requirements
- Aesthetic: Kawaii, Cottagecore, Pastel, Hand-drawn
- Fonts: Fredoka (Headings), Nunito (Body)
- Palette: Soft Pinks, Sage Greens, Creamy Yellows, Sky Blues

## Implementation Steps
1. Define CSS variables for the color palette.
2. Setup base typography styles.
3. Design common UI components (Hearts, Coin counter, Buttons).
4. Define the Capybara character style.

## Success Criteria
- Cohesive look across all 3 levels.
- Accessible color contrast.
- Friendly, playful feel.
